# url64

url64 is a library to generate really unique and url friendly IDs based on UUID

## Usage

```
user=> (url64.core/generate-id)
"U7PVVMkSSJGepn8RKilg-A"
```

leiningen:

```
[url64 "1.0.0-SNAPSHOT"]
```
